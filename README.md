# TransFi Frontend Setup (Next.js)

## Overview
The frontend uses Next.js for the UI (signup, login, dashboard, QR payments) and Web2 backend (API routes for fiat, OTP, authentication). It features a glassy design with gold (#D4A017), brown (#4A2C0F), cream (#F5E8C7), black (#1A1A1A), and white (#FFFFFF) colors, optimized for mobile and slow internet in East Africa.

## Prerequisites
- **Node.js**: v18+ (install from [nodejs.org](https://nodejs.org)).
- **Vercel CLI**: `npm i -g vercel`.
- **Text Editor**: VS Code recommended.
